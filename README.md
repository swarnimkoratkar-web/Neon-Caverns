## *Project Solution*

*Neon Caverns* offers a refreshing alternative by providing a *simple, enjoyable, retro-style endless runner* experience. The game focuses on:

* *Pure gameplay* without overwhelming competitive elements
* *Smooth endless runner mechanics*
* *Neon-themed visuals* with bloom lighting for aesthetic immersion
* *Simple pick-up-and-play design* suited for casual mobile players

This gives users a relaxing, fun experience rather than a high-pressure one.
